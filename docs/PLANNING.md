**5분 컷 삼국지 카드 배틀** - 부스터팩 뜯는 쾌감 + AI가 만드는 매번 다른 전장

## 🎁 카드 부스터 시스템

### 첫 시작
- 게임 시작 → 스타터 부스터 3팩 지급
- 부스터 찢는 애니메이션 (손가락으로 스와이프해서 뜯기!)
- 카드가 한 장씩 뒤집히며 공개 → 등급별 이펙트

### 부스터팩 종류
| 팩 | 내용 | 획득 방법 |
|----|------|-----------|
| 🟢 일반팩 | 카드 3장 (⭐ 확정) | 전투 승리 |
| 🔵 희귀팩 | 카드 4장 (⭐⭐ 1장 확정) | 연승 3회 / 일일미션 |
| 🟣 영웅팩 | 카드 5장 (⭐⭐⭐ 1장 확정) | 시나리오 클리어 |
| 🌟 전설팩 | 카드 5장 (🌟 확률 UP) | 특별 이벤트 |

### 뽑기 연출
1. 팩 선택 → 포장지 찢기 (스와이프/탭)
2. 카드 뒷면 나열
3. 탭하면 한 장씩 뒤집힘
4. 등급별 연출: ⭐ 기본 / ⭐⭐ 빛남 / ⭐⭐⭐ 홀로그램 / 🌟 금빛 폭발
5. 효과음 + 화면 이펙트

### 뽑기 확률 (일반팩 기준)
- ⭐ 70% / ⭐⭐ 22% / ⭐⭐⭐ 7% / 🌟 1%

## 🤖 AI 차별점

### 1. AI 생성 전장 이벤트
- 매 배틀마다 전장 조건 생성
- 예시: "폭우 → 화공 무효화", "산악 → 방어+2", "야간 기습 → 첫 턴 전위 스킵"
- 같은 덱이라도 전장에 따라 전략이 달라짐

### 2. AI 적응형 적군
- 플레이어 덱/전략 분석 → 카운터 덱 구성
- 항상 아슬아슬한 승부

### 3. AI 시나리오 모드
- "적벽대전", "호로관 전투" 등 역사 시나리오 자동 생성
- 특수 승리 조건 + 제한 카드 + 스토리

## 🃏 카드 시스템

### 무장 카드 스탯 (4개)
| 스탯 | 역할 |
|------|------|
| ⚔️ 무력 | 물리 공격력 |
| 🛡️ 통솔 | 체력 (HP) + 물리 방어력 |
| 🧠 지력 | 스킬 공격력 + 스킬 방어력 |
| 🏰 방어 | 받는 데미지 고정 감소 |

### 등급
| 등급 | 예시 | 특징 |
|------|------|------|
| ⭐ 일반 | 장합, 문추, 진란, 기령, 반장 | 기본 스탯 |
| ⭐⭐ 희귀 | 황충, 장료, 감녕, 서황, 방덕 | 스킬 1개 |
| ⭐⭐⭐ 영웅 | 관우, 조조, 손권, 유비, 장비 | 스킬 2개 + 패시브 |
| 🌟 전설 | 여포, 제갈량, 사마의, 주유 | 고유 스킬 + 궁극기 |

## ⚔️ 배틀 시스템

### 레인 배치
```
[전위] [중위] [후위]  vs  [전위] [중위] [후위]
```

### 덱 구성
5장: 무장 카드 3장 + 전법 카드 2장

### 턴 진행
1. 양측 동시에 전법 카드 사용 (선택, 핸드에서 1장)
2. 전위부터 순서대로 공격 (전위 → 중위 → 후위)
3. 상대도 전위부터 공격
4. 전위가 쓰러져야 중위/후위 타격 가능

### 데미지 계산
```
물리 데미지 = 무력 - 방어 (최소 1)
스킬 데미지 = 지력(공격측) - 지력(방어측) × 0.5 (최소 1)
HP = 통솔 × 3
```

### 승리 조건
- 상대 무장 3체 전멸
- 3턴 후 잔여 HP 합산 비교

### 전략 포인트
- 전위에 탱커(높은 통솔+방어) → 중후위 보호
- 전위에 딜러 → 빠른 돌파 but 금방 쓰러짐
- 후위에 지력형 → 전법 카드 시너지 극대화
- AI 전장 조건에 따라 매번 배치 변경 필요
